# exam_preparation
A repository to keep track of my exam preparation and my level of confidence with School 42's exam scheme. For School 42.

## DESCRIPTION
Inside this repo you will find directories named ranks, from rank 02 up to a max of rank 06.  

Each rank* directory will have sub-directories of each level from that exam, between level00 and level03, each of these will contain code I have written in practice or mock exams. Later I will include directories of each practice exam I take, to track my progress or lack there of.
